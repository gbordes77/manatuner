Identity

“The only mana calculator that counts your dorks & rocks.”
ManaTuner is a free Magic: The Gathering manabase calculator. It computes per-spell cast probabilities accounting for lands, mana dorks, mana rocks, rituals, and acceleration. The visible interface combines a Material-UI component layer with parallel CSS variables, both shipped together.
i
Authenticity

The mana palette is the canonical Wizards colour set — no pastel-ization, no desaturation.

ii
Knowledge tier

The Library uses a blue→purple gradient normally reserved for high-tier products.

iii
Quiet UI, loud subject

Cinzel serifs for headings, Roboto cascade for body, parchment ground for surface.
